Nacha Gem

A robust and compliant ACH (Automated Clearing House) file parser for Ruby.

Overview

The Nacha gem provides a comprehensive toolkit for developers working with ACH files in Ruby applications. It allows for easy parsing of existing ACH files, generation of new files from scratch, and validation against NACHA operating rules to ensure compliance.

Key Features

  • Parse ACH files into structured Ruby objects.
  • Build and generate compliant ACH files.
  • Validate records and files against NACHA rules.
  • Support for various Standard Entry Class (SEC) codes.

Resources

Installation

Add this line to your application's Gemfile:

gem 'nacha'

And then execute:

$ bundle install

Or install it yourself as:

$ gem install nacha
